
# Version 4

## Funktionaliteter

EDIT*- Söka efter program som innehåller söktermen i sitt programnamn, land eller stad.
*- Få upp alla program som sökresultat på söksidan

*- Möjlighet att sortera sökresultat i stigande/fallande ordning efter:
    - Bokstavsordning
    - Antagningspoöng

*- Jämför-söksida
    *- Sök efter programnamn och lägg till i en lista
    *- Valda programs information visas i en tabell
    - Förslag utifrån de program som sparats från söksidan

*- En detaljsida för ett program när man trycker på "läs mer".

EDITEDIT*- Randomiserade "Land"-kort på indexsidan. Gör en automatisk sökning på landet (i sökfunktionen) när man trycker "visa program".

- Möjlighet att filtrera sökning efter:
    - Nivå
    - Antagningspoöng
    - Språk
    - Visumkrav
    - Soldagar

- Möjlighet att spara ett program från sökresultatet i "favoriter".

- En sida som visar alla "favoriter"
    - Möjlighet att ta bort program från "favoriter".
